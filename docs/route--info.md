## `/info`
Returns information about the plugin and the current Obsidian instance.

| Parameter    | Value   | Optional? | Description |
| ------------ | ------- |:---------:| ----------- |
| `call-id`    | string  |    ✅     |             |
| `debug-mode` | boolean |    ✅     |             | 
| `x-error`    | string  |           |             |
| `x-success`  | string  |           |             |
| `action`     | string  |           |             |
| `vault`      | string  |           |             |

=> HandlerInfoSuccess
